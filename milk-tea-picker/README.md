# 今天喝什么奶茶

一个离线可用的奶茶随机推荐网页，支持：

- 城市级筛选（先选地区，再选城市）
- 每个城市热门 TOP5 标注与点选（可锁定某个 TOP 项）
- 仅在 TOP5 中抽取模式（未选城市时会提示）
- 品类筛选 + 冷热筛选
- 品牌搜索与多选
- 今日固定推荐 / 再抽一杯
- 每次抽取自动生成暖心简短赠语（结果区显眼展示）

## 本地使用

1. 打开 `D:/code/milk-tea-picker/index.html`
2. 直接在浏览器中使用（无需安装依赖）

## 数据说明

- 基础饮品库：`D:/code/milk-tea-picker/data.js`
- 城市映射、热榜算法、品牌扩充逻辑：`D:/code/milk-tea-picker/app.js`

当前估算规模：109 个品牌、约 515 款饮品、86 个城市（可继续扩展）。
