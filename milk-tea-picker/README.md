# 今天喝什么奶茶

一个离线可用的奶茶随机推荐网页，支持：

- 全国省市两级筛选（先省份，再城市）
- 覆盖中国全部省级单位与城市条目（省级 34，城市/地区/自治州等 393）
- 每个城市热门 TOP5 标注与点选（可锁定某个 TOP 项）
- 仅在 TOP5 中抽取模式（未选城市时会提示）
- 城市快速搜索
- 品类筛选 + 冷热筛选
- 品牌搜索与多选
- 今日固定推荐 / 再抽一杯 / 来点惊喜
- 避免重复推荐开关 + 近期随机抽取记录
- 每次抽取自动生成暖心简短赠语（结果区显眼展示）

## 本地使用

1. 打开 `D:/code/milk-tea-picker/index.html`
2. 直接在浏览器中使用（无需安装依赖）
